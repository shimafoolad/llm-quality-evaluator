# Sample Data

This folder contains sample data files for testing the response quality evaluator.

## JSON Files
- `json_files/sample_conv_1.json` and `sample_conv_2.json`: Sample conversation data in JSON format with chat history, summaries, and participant information.

## CSV Files
- `csv_files/sample_data_1.csv` and `sample_data_2.csv`: Sample evaluation data with input prompts, responses, and metadata.
- `convo_files/sample_convo_1.csv`: Sample conversation data in CSV format.

These files contain dummy/anonymized data for demonstration and testing purposes. Replace with real data as needed for actual evaluations.
